DRAW THINGS PROMPTS — Intimacy Position Silhouettes
====================================================

Use these in Draw Things (free, Mac App Store).
Settings: 512x512 or 768x512 (landscape), 25-30 steps, CFG 7-8

NEGATIVE PROMPT (use for ALL):
face, eyes, fingers, detailed, realistic, 3d, gradient, outline, line art, text, watermark, signature, anatomy details, background objects


BODY TYPE MODIFIERS (append to any prompt):
- average: (no modifier needed)
- curvy: "curvy full figured bodies"
- athletic: "athletic muscular toned bodies"
- slim: "slim thin slender bodies"
- plus: "plus size larger heavier bodies"

COUPLE VARIANT MODIFIERS:
- him-her: "man and woman" (default in prompts below)
- her-her: replace "man" with "woman" / "both women"
- him-him: replace "woman" with "man" / "both men"

====================================================
INKSCAPE WORKFLOW (after generating each PNG)
====================================================
1. File > Import > select the PNG
2. Path > Trace Bitmap > Brightness cutoff (~0.45)
3. Click OK, then delete the original PNG layer
4. Object > Ungroup (repeat until fully ungrouped)
5. Select figure A paths > Object > Group > rename to "figureA" in XML editor
6. Select figure B paths > Object > Group > rename to "figureB" in XML editor
7. Select all > remove Fill color (will be set by code)
8. File > Document Properties > set width=400, height=250
9. File > Save As > Plain SVG
10. Name: {id}-{variant}-{bodytype}.svg (e.g. ip001-him-her-average.svg)

## 🔄 Update Process

### When to Update

**Terms of Service:**
- New features added
- Policy changes
- Legal requirements change
- Quarterly review

**Privacy Policy:**
- Data practices change
- New services added
- Legal requirements change
- Quarterly review

**FAQ:**
- New features launched
- Common questions arise
- User feedback received
- As needed

### How to Update

1. **Draft Changes**
   - Review current document
   - Identify necessary changes
   - Draft new version
   - Legal review (if major changes)

2. **Version Control**
   - Update version number
   - Update "Last Updated" date
   - Document changes in history
   - Archive previous version

3. **User Notification**
   - Email for material changes
   - In-app notification
   - Update log in app
   - Grace period before enforcement

4. **Implementation**
   - Update documents in app
   - Update website
   - Update app store listings
   - Train support team

# Between Us — Brand Guardrails

> Internal reference for anyone writing copy, UI text, error messages,
> notifications, or marketing material. When in doubt, read this first.

---

## The Core Feeling

Between Us is a **quiet place** two people share. It's not a tool,
not a tracker, not a coach. It's the digital equivalent of lying in bed
together, phones down, just… talking. Every word we write should feel
like it belongs in that space.

**Brand line:** *Where closeness deepens.*

---

## Voice Principles

| Principle | What it means | Example |
|-----------|--------------|---------|
| **Warm, never clinical** | We never sound like a therapist's worksheet. | ✅ "Your story starts here" · ❌ "No entries found" |
| **Intimate, never performative** | We don't cheer or gamify. Connection isn't a sport. | ✅ "3 nights connected" · ❌ "Streak 🔥 Keep it up!" |
| **Present, not pushy** | We suggest, never demand. The couple decides. | ✅ "Whenever you're ready…" · ❌ "Don't forget to complete today's prompt!" |
| **Confident quiet** | Short sentences. Let the space breathe. | ✅ "Something beautiful awaits ✨" · ❌ "AMAZING! You both answered!" |
| **Honest, not salesy** | Premium features are invitations, not gates. | ✅ "Discover deeper moments" · ❌ "UPGRADE NOW to unlock!" |

## Between Us ALWAYS…

- Uses **first names or "love"** — never "User" or "you"
- Treats the **couple as equals** — partner's premium unlocks for both
- Puts **privacy front and center** — encrypted by default, no cloud without consent
- Feels **handcrafted** — like someone cared about every word
- Respects **silence** — not every screen needs a subtitle
- Celebrates **small moments** — a quiet journal entry matters as much as a milestone

## Between Us NEVER…

- Uses **gamification language**: streak, points, score, level up, achievement unlocked, leaderboard, challenge, badge, rank
- Uses **clinical/analytical language**: relationship health, engagement score, churn, retention, conversion, funnel, optimization
- **Shames inactivity**: "You haven't journaled in 3 days!" → Instead, stay quiet or say "Welcome back 💜"
- Uses **ALL CAPS** for excitement (except subtle eyebrow labels like "TODAY'S PROMPT")
- Shows **countdown timers** or urgency tactics for purchases
- Uses **manipulative dark patterns**: "Are you sure you want to miss out?"
- Sends **guilt-based notifications**: "Your partner is waiting…" with a red badge
- **Compares couples**: no rankings, no "other couples also…", no benchmarks
- Uses the word **"user"** in any visible UI

## Visual Tone — Velvet Intimacy Palette

- **Dark theme only** — Near-black (#080508) background, warm candlelight cream text (#F5EAE6)
- **Midnight velvet surfaces** (#130D17) — cards, inputs, elevated UI elements
- **Dark aubergine** (#1B1221) — secondary surfaces, subtle elevation
- **Elevated velvet** (#231929) — tertiary surfaces, modal overlays
- **Sexy red accent** (#C3113D) — primary CTAs, active states, key interactions
- **Deep berry-red accent** (#A00D31) — secondary accent, links, subtle highlights
- **Wine-deep** (#4E0820) — gradient endpoints, deep accent
- **Antique amber gold** (#C49A6A) — **premium-only**, never in main UI. Reserved for Pro badge, paywall, milestones.
- **Plum vignette** (#1A0C1E) — atmospheric gradient backgrounds
- **No teal, no green, no bright pink** — the palette is velvet plum, deep rose-wine, and candlelight cream only
- **Glass/blur effects** — plum-tinted velvet glass `rgba(27,18,33,...)`, everything feels layered and warm
- **Borders** — white at very low opacity: `rgba(255,255,255,0.04–0.07)` — never raw white, never colored
- **No hard borders** — rounded corners (cards: 20, buttons: 16, pills: 999), subtle gradients, breathing space
- **Shadows**: Dramatically reduced — max opacity 0.12, mostly 0.04–0.08
- **Animations**: Gentle. Slow fades (250–1000ms), soft springs (damping 20–25). Never bouncy or attention-seeking.

| Role | Font | Size | Weight |
|------|------|------|--------|
| **Display** | DM Serif Display | 36px | 400 (Regular only) |
| **H1** | DM Serif Display | 32px | 400 |
| **H2** | DM Serif Display | 28px | 400 |
| **H3 / Section label** | Lato Bold | 16px | 600 |
| **Body** | Lato Regular | 15px | 400 |
| **Caption** | Lato Regular | 13px | 400 |
| **Button** | Lato Bold | 16px | 600 |
| **Label / Eyebrow** | Lato Bold | 12px | 600, uppercase, letter-spacing 1.5+ |
| **Pull Quote** | DM Serif Display | 22px | 400, italic |
| **Micro** | Lato Regular | 12px | 400 |

**Rules:**
- **Serif (DM Serif Display) is ONLY for**: display titles, h1, h2, pull quotes. Never for body, labels, buttons, or captions.
- Serif headings are never bold (DM Serif Display only has Regular weight)
- Maximum font weight in the app is 700 — prefer 600 for emphasis
- Headings: 28–36px · Body: 15px · Labels/captions: 12–13px
- Border colors: `rgba(255,255,255, 0.04–0.08)` — never colored, never raw white
- Background tints use plum: `rgba(28,21,32, 0.15–0.60)` — never teal, never raw white

## Premium Philosophy

Premium in Between Us is an **invitation to go deeper**, not a wall.
Free users can preview a taste of the experience — 3 read-only prompts across
heat levels 1-3. Premium unlocks everything: unlimited prompts, responses,
love notes, calendar, partner connection, cloud sync, and ongoing new content.

**Never say:**
- "You've reached your limit"
- "Upgrade to continue"
- "You can't do that"

**Instead say:**
- "Discover the full experience"
- "There's more waiting for you"
- "This is part of the deeper journey"
- "Premium unlocks responding, saving, sharing, and everything else"

---

## Notification Philosophy

Notifications are a privilege, not a right. Every notification must pass
this test: **"Would I want to receive this while lying next to my partner?"**

If the answer is no, don't send it.

- Maximum 1 notification per day
- Never guilt-based
- Always dismissable
- Never reveal private content in the preview
